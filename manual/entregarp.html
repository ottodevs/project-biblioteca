¿Cómo entregar un prestamo?

Para realizar la operación de entregar un libro en prestamo existen dos forma de realizar dicha operación:

  1. Mediante el panel izquierdo Prestamo -> Entregar
  2. Mediante el menu Operaciones -> Prestamo-> Entregar (Como se muestra en la siguiente figura)

Figura: Menu Operaciones -> Prestamo

Una vez seleccionada la opción de entregar, debe llenar el formulario que se le muestra en la siguiente figura. Una vez completado el formulario, le damos al boton entregar, si el libro no se encuentra en prestamo aparecera un error en caso contrario se realizará la entrega del libro satifactoriamente.
