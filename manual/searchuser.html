¿Cómo buscar un usuario?

Para realizar la operación de buscar un usuario existen dos forma de realizar dicha operación:

  1. Mediante el panel izquierdo Usuarios -> Buscar
  2. Mediante el menu Operaciones -> Usuario -> Buscar (Como se muestra en la siguiente figura)

Figura: Menu Operaciones -> Usuario

Una vez seleccionada la opción de buscar, debe llenar el formulario que se le muestra en la siguiente figura. Terminado de completar el formulario le damos al boton buscar y los datos correspodiente al usuario será mostrado en cada uno de los campos, en esta opción solo se le permitira visualizar los datos del usuario, si desea editar alguna información del usuario vea ¿Cómo editar un usuario?.
