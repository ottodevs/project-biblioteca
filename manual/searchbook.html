¿Cómo buscar un libro?

Para realizar la operación de buscar un libro existen dos forma de realizar dicha operación:

  1. Mediante el panel izquierdo Libros -> Buscar
  2. Mediante el menu Operaciones -> Libro-> Buscar (Como se muestra en la siguiente figura)

Figura: Menu Operaciones -> Libro

Una vez seleccionada la opción de buscar, debe llenar el formulario que se le muestra en la siguiente figura. El campo que debe completar es el tipo de búsqueda se eligen dentro de la caja de opciones, estas son:

  • Cota:  Busca el libro por medio de la cota.
  • Autor: Busca el libro por medio del nombre y/o apellido, la busca se realiza por la coincidencia que se encuentra de estas.
  • Título: Busca el libro por medio del titulo, la busca se realiza por la coincidencia que se encuentra de esta.
  • Materia: Busca el libro por medio de la materia, la busca se realiza por la coincidencia que se encuentra de esta (como se muestra en la siguiente figura).

En esta opción solo se le permitira visualizar los datos del libro, si desea editar alguna información del libro vea ¿Cómo editar un libro?.
