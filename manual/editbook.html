¿Cómo editar un libro?

Para realizar la operación de editar un libro existen dos forma de realizar dicha operación:

  1. Mediante el panel izquierdo Libros -> Editar
  2. Mediante el menu Operaciones -> Libro-> Editar (Como se muestra en la siguiente figura)

Figura: Menu Operaciones -> Libro

Una vez seleccionada la opción de editar, debe de buscar el libro y llenar el formulario que se le muestra en la siguiente figura. El formulario aparecera completado con los datos del libro que se desea editar.

Editando el dato o los datos que se desean cambiar, continuamos dandole al boton editar y los datos correspodiente al libro seran editando satifactoriamente. Si desea visualizar los datos recientemente actualizado vea ¿Cómo buscar un libro?.
